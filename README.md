# NexPath - Career Path Generator

An AI-powered full-stack web application that helps users identify their career path, improve skills, access learning resources, and track progress with visual analytics.

## 🚀 Features

- 🔐 Secure Authentication System
- 🎯 40+ Career Roles (Tech & Arts)
- 📚 Personalized Learning Roadmaps
- 📊 Visual Progress Tracking with Charts
- 🎥 Integrated Learning Resources (Notes, Videos)
- 💼 Job Portal Links
- 📈 Weekly Progress Analytics

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Chart.js for visualizations
- React Router for navigation
- Axios for API calls

**Backend:**
- Node.js with Express
- MongoDB with Mongoose
- JWT Authentication
- bcrypt for password hashing
